# randomForestSRC: 

## Random Forests for Survival, Regression and Classification (RF-SRC)

A unified treatment of Breiman's random forests for survival, regression and classification problems based on Ishwaran and Kogalur's random survival forests (RSF) package. The package runs in both serial and parallel (OpenMP) modes.

This GitHub repo is a fork of randomForestSRC package release on CRAN. This repo is used to track changes and test new versions of the package.
